Deployment notes — FHIR SMART & webhooks
Internal working tool — not for clinical use without pharmacy and informatics committee review. Calculator is display-only and informational; it does not place orders or document administration. mL volume is never displayed.

Order parameters

Advanced — result webhook (optional)
When set, each calculated dose is POSTed as JSON. In a SMART deployment this is replaced by a CDS Hooks / documentation call. Leave blank to keep everything on-device.

Which order?

Choose the meal, snack, or bedtime order you are dosing. The icon appears top-left as a cue that the right order was selected. Changing the order requires re-confirming the order details below.
